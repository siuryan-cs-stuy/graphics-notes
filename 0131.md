# Peering into the depths of color

## Color Depth
The amount of data used to represent a single pixel

| size  | color options                             |
|-------|-------------------------------------------|
| 1 bit | 1 color, on/off                           |
| 2 bit | 1 color, on/off, plus 1 bit for intensity |
| 3 bit | RGB, on/off                               |
|       |                                           |
|       |                                           |
